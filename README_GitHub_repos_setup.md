# Instructor Use Only

1. Create a Google Drive folder for the course.
    - not all subfolders in this folder will be Git repos
    - name: DA2-main

2. Create new repos from GitHub Desktop:
    - File => New Repository
    - name: 
           course-setup
           teaching-log
           applied-solutions
           homework-log
           homework-solutions
           classwork-log

    - local path for all repos: Google Drive/DA2-main
    - initialize each with a README file
