# Course-Setup Instructions

## Windows
**1. Check for existing software:**

Open Terminal & type:

* *python --version*
* *py -0* (to check if you have multiple Python versions)
* *pip --version* (only if you have Python; pip is a tool that manages Python packages and is used to install Python libraties that do not come with Python installation)
* *git --version*
* *code --version*

**NOTE:** Pin terminal to taskbar.


**2. Install the latest version of Python (3.14.2)**

* https://python.org
* Use standalone installer, **not** Python Installation Manager
* Make sure to check **[x] Add python.exe to PATH** (this allows you to run Python from any terminal)
* Verify in the terminal that you have it: *python --version*

**3. Install & Setup VS Code**

* Search for code.visualstudio.com - (https://code.visualstudio.com/docs/setup/windows)
* Verify: *code --version*
* Pin to taskbar
* Setup VS Code:
    - Install **extensions:**
          *Python*, *autopep8*
    - Go to **File => AutoSave**
    - Go to **Manage => Keyboard Shortcuts => Run Python File** and add a shortcut **Ctrl + R**

**4. Install Git**

* Search for git-scm - (https://git-scm.com/install/windows)
* At the end of installation, check **[x] Launch Git Bash** (this opens Git terminal  where you can instantly verify Git installation and manually configure Git, if needed)

**5. Create a GitHub account**

* This is your remote GitHub

**6. Download GitHub Desktop**

* Login to your remote GitHub
* Configure Git automatically, if asked
* Pin to taskbar
